# Education Quiz Platform – Demo Video Script

Use this script for voiceover (your voice or AI). Adjust pacing to match what you show on screen.

---

## Opening

This is a platform created especially for kids to improve their subject knowledge. Once you open the webpage, it looks like this. You’ll see different subjects—at the moment there are six: Mathematics, Science, English, Sinhala, Social Studies, and Buddhism.

---

## How it works – Subject and topics

Let me show you how it works. When you select a subject, you’re taken to a page where you see different subsections that cover the syllabus. The kid can select several sections or select all. Once they’ve chosen, they click to start the quiz.

---

## The quiz – Questions and feedback

The quiz picks up to 10 random questions from the selected sections. Here we’re doing Multiplication. If the answer is correct, it moves to the next question. If the answer is incorrect, it shows that it’s wrong, shows the correct answer, and explains how to get it—the method. So when the kid gets something wrong, they can learn from the explanation.

---

## Timer (optional)

If the kid wants to set a timer for each question, that’s possible too. They have to answer within that time, which also helps improve their thinking speed.

---

## Finishing a quiz

There are 10 questions. Once all 10 are done, the platform shows the score. Then the kid can choose the same topics again or pick new topics. Let’s select a few other topics and start another quiz—the questions will only come from these sections.

---

## Another subject – Science

Let’s try another subject. I’ve selected Science. Inside Science there are different sections as well. I’ll select a few subsections and start the quiz. When I choose a wrong answer, it shows the correct answer and the explanation, so kids learn from their mistakes.

---

## Progress – The important part

One of the most important features is progress. The platform tracks how many questions the student attempted, how many were correct and incorrect, and shows the percentage. It highlights key areas to improve—for example, Addition might be an area to improve. The kid can then click **Practice** and go straight to questions from that section only. That’s one of the key strengths of this platform.

---

## Optional short closing (if you want an outro)

So that’s the Education Quiz Platform—built for kids to practice, learn from mistakes, and improve in all six subjects. Thank you for watching.
