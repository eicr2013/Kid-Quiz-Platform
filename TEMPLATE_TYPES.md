# Question Template Types 🎯

## ✅ FULLY VARIABLE Templates (All numbers change)

### Type 1: **a + b = ?** (BOTH variables)
```
What is {num1} + {num2}?
Examples:
- What is 7 + 3?
- What is 15 + 23?
- What is 45 + 38?
```

### Type 2: **a - b = ?** (BOTH variables)
```
What is {num1} - {num2}?
Examples:
- What is 20 - 5?
- What is 67 - 29?
- What is 400 - 175?
```

### Type 3: **a × b = ?** (BOTH variables)
```
What is {num1} × {num2}?
Examples:
- What is 3 × 5?
- What is 7 × 8?
- What is 9 × 6?
```

### Type 4: **a + ___ = z** (ALL THREE variables!)
```
{num1} + ___ = {total}
Examples:
- 15 + ___ = 50
- 23 + ___ = 70
- 35 + ___ = 80
```

### Type 5: **a - ___ = z** (ALL THREE variables!)
```
{total} - ___ = {num1}
Examples:
- 60 - ___ = 25
- 80 - ___ = 47
- 100 - ___ = 63
```

### Type 6: **a + b + c = ?** (THREE numbers!)
```
What is {num1} + {num2} + {num3}?
Examples:
- What is 15 + 20 + 10?
- What is 25 + 30 + 15?
- What is 12 + 18 + 22?
```

### Type 7: **___ × b = z** (Missing first number)
```
___ × {num2} = {total}
Examples:
- ___ × 5 = 35
- ___ × 7 = 56
- ___ × 4 = 32
```

### Type 8: **a - ___ = z** (Subtraction blank)
```
{total} - ___ = {result}
Examples:
- 60 - ___ = 35
- 90 - ___ = 52
- 75 - ___ = 28
```
