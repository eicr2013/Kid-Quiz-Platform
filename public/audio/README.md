# Audio files for the quiz app

Add these files in this folder (`public/audio/`). The app will use them automatically.

| File name | When it plays | Format |
|-----------|----------------|--------|
| **background-music.mp3** | Loops in the background on the app (can be muted in Settings) | MP3 preferred |
| **button-click.mp3** | When user clicks subject, category, Start Quiz, answer options, Next Question, etc. | Short (e.g. 0.1–0.5 s) |
| **quiz-complete.mp3** | When the user finishes the quiz (results screen) | Short fanfare / success |
| **encouragement-low-score.mp3** | When the user finishes with **fewer than 5 correct answers** (encouragement) | Short, positive message |

**Naming:** Use the exact names above so the code can find them.  
**Formats:** MP3 is most widely supported. OGG also works in many browsers.

After you add the files here, the app will play them at the right moments.
